How to Use the System  

The CareHome Energy System is a frontend-based system that simulates energy usage in a care home.  
It includes three roles: Resident, Manager, and Admin.  

To use the system:  
• Open the application in a browser  
• Select a role (Resident / Manager / Admin)  
• Enter the login details  
• You will be redirected to the dashboard

What Each Role Can Do  

Resident  
• View personal energy usage  
• Turn devices ON/OFF  
• View charts and AI prediction  
• See leaderboard  
• Request devices  

Manager  
• View all residents  
• Monitor energy usage  
• Use facility timer  
• View energy graphs  

Admin
• Manage users (add/remove/change roles)
• Manage devices
• View system analytics
• Change system settings
• Reset system data
